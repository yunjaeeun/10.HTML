폼 관련 태그

form 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그이다.
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 한다.
* action 속성: 폼의 입력된 값들을 전송받을 서버의 주소를 입력한다.
* method 속성: get / post 방식으로 전송 방식을 지정한다.(Http method 방식 지정)

 

submit 버튼을 눌렀을 때 action에 지정한 경로로 method에 지장한 방식에 따라
input 태그에 입력한 값들을 전달한다.

숫자와 관련된 input 태그

type="number"

- 입력창에 숫자를 입력하는 것이 기본이지만, 브라우저에 따라 스핀박스가 표시 되기도 한다.(크롬은 스핀 박스가 보인다.)


type="range"

- 슬라이드바를 통해 숫자를 지정할 수 있다.

textarea 태그

input type="text"와 비슷하지만 여러 줄을 입력할 수도 있고
input 태그와 달리 세로나 가로로 크기 조절이 가능하다
(css 속성인 resize로 크기 조절을 고정할 수 도 있고 범위를 벗어나면 스크롤 바가 생긴다.)
(추후 우리가 javascript를 통해 textatrea에 있는 값을 다룰 때는 innerHTML/innerTEXT가 아닌 value라는 개념으로 들어있는 값을 다루게 된다.)
